# 🚀 Codiva - Soluciones Web para Negocios

## 🚀 Desarrollo Local

Para ejecutar este proyecto localmente:

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```
